🔧 Firebase Setup — گیٹ لاگ سسٹم

Connect your free Firebase database. Do this once — all staff share the same live data.

1
Create Firebase project

Go to console.firebase.google.com → Add project → name it (e.g. farm-gate) → disable Analytics → Create.

2
Enable Realtime Database

Build → Realtime Database → Create database → any region → Start in test mode → Enable.

3
Register Web App & copy config

Project Overview → Add app → Web → Register → copy the 4 values from firebaseConfig.

Please fill in all four fields.
